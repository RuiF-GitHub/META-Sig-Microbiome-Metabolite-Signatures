# Results & reporting

Guidelines for presenting outputs:

- Top N microbially derived candidates with evidence breakdown
- Volcano/Manhattan plots for enrichment analyses
- Networks/heatmaps for metabolite–taxon/function associations
- Export formats: TSV/CSV with columns for ID, evidence, scores

> Consider adding a `/data` or `/notebooks` folder in your code repository and link results here.
