# Getting started

This project website is generated with **GitHub Pages** using the **Just the Docs** theme. To deploy **META‑Sig: Microbiome Metabolite Signatures**:

1. Create a new GitHub repository named:
   - `META-Sig-Microbiome-Metabolite-Signatures`
2. Upload all files in this folder to the **repository root**.
3. In the repository, go to **Settings → Pages** and set:
   - **Source**: *Deploy from a branch*
   - **Branch**: `main` (or your default) and **Folder**: `/ (root)`
4. Save. After ~1–2 minutes, your site will be live at:
   - `https://<your-username>.github.io/META-Sig-Microbiome-Metabolite-Signatures/`

> Tip: Update `_config.yml` → `aux_links` to your real repo URL, and set `url`/`baseurl` once the site is live.

## Local editing (optional)
If you want to preview locally with Jekyll, install Ruby/Jekyll and run:
```bash
bundle install
bundle exec jekyll serve
```
*Optional — GitHub builds the site for you.*
